Searching the User's Guide for the WN Server

You can search the documentation and view the matches to your search term in one line of context. If you select one of the highlighted phrases in the context lines you will be taken to that point in the document. Try this out, it is one of the nice features of WN. Searches are not case sensitive.

Note: This is not an indexed search engine and requires no indexing of documents. You cannot use boolean operations but you can use regular expressions.
